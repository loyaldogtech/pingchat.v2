# PingChat

PingChat is a real-time agency and client project chat application built with ASP.NET Core.

## Tech Stack

- C#
- ASP.NET Core
- Razor Pages
- ASP.NET Core Identity
- Git / GitHub

## Current Status

This project is in active development.

## Goal

The goal of PingChat is to provide a lightweight collaboration app for agencies and clients, with channels for internal discussion and client-facing project communication.

## Why I Built This

I built PingChat to strengthen my full-stack development skills in ASP.NET Core and to create a real, publishable project that demonstrates authentication, real-time communication, and business-focused application design.

## Planned Features

- User registration and login
- Agency and client roles
- Channel-based chat
- Real-time messaging with SignalR
- Persistent message history
- Role-based access control
- Public deployment
